# 思い出機能

## コンセプト

遠征の記録を、後から楽しく振り返られる機能。

「管理アプリ」と「アルバム」を組み合わせたような体験を目指す。

---

## 思い出を見る

場所の詳細画面から「思い出を見る」を開ける。

例

```
Liquem

★★★★★ お気に入り

訪問回数　5回

最後に行った日
2028/03/15

────────────

📖 思い出を見る ▶
```

---

## 思い出一覧

訪問した新しい順に表示する。

例

```
2028/03/15
春東京遠征

📷 写真4枚

📝 メモあり

────────────

2027/11/21
舞台○○遠征

📷 写真6枚

📝 メモあり
```

---

## 思い出詳細

表示内容

- 訪問日
- 遠征情報
- 写真
- メモ

例

```
2028/03/15

春東京遠征

📷 写真

・外観
・購入品
・店内

📝 メモ

限定アクセサリー購入！

15時頃は空いていた。

また行きたい。
```
